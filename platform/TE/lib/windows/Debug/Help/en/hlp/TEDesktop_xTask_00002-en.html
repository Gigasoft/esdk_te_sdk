Downloading and Installing

This topic describes how to download and install the TE Desktop.

Open a browser. In the address box, enter http://e.huawei.com to access the Huawei Enterprise website. In the search box, enter TE Desktop to search for the software. Follow the onscreen instructions to download and install the TE Desktop. To uninstall the TE Desktop, choose Start > All Programs > TE Desktop > Uninstall TE Desktop and follow the onscreen instructions.

OPEN SOURCE SOFTWARE NOTICE: Access to the TE Desktop installation directory and view the detailed information about OPEN SOURCE SOFTWARE NOTICE.
