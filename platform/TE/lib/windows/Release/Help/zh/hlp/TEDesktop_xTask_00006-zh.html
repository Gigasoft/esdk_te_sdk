设置

包括个人信息、基本设置、媒体设置、呼叫和演示设置、修改密码。

您可以通过以下三种方式进入设置界面：
  • 在TE Desktop主界面左上角单击选择“设置”。
  • 在TE Desktop主界面左下角单击。
  • 在TE Desktop主界面单击头像。

个人信息

进入个人信息界面可以查看帐号、SIP URI、IP地址和端口。

基本设置

  • 主窗口设置：设置主窗口的显示状态，设置后立即生效。
  • 登录设置：设置TE Desktop是否随Windows操作系统启动而自动启动和自动登录，以及设置登录后默认首页，重启后生效。使用免帐号登录的用户不能设置自动登录、记住密码和自动启动。
  • 语言设置：设置TE Desktop使用的语言，重启后生效。
  • 铃声设置：设置来电时本端听到的铃声，设置后立即生效。
    • 单击选择框从下拉框中选择系统自带铃声。
    • 单击播放所选铃声。

媒体设置

  • 音频输入：单击“音频输入”下方的选择框为TE Desktop选择音频输入设备。单击可对麦克风进行测试。
  • 音频输出：单击“音频输出”下方的选择框为TE Desktop选择音频输出设备。单击可对扬声器进行测试。
  • 视频设备：安装视频输入设备后，才可配置视频设备参数。

    单击“视频设备”下方的选择框为TE Desktop选择视频输入设备。单击可对摄像头进行测试。

    • 画质优先：在带宽不变的情况下，优先保证图像分辨率。
    • 流畅优先：在带宽不变的情况下，优先保证帧率。

单击“确定”，立即生效。通话过程中不支持媒体设置。

呼叫和演示设置

  • SRTP加密：选择是否对媒体流进行加密。
    说明：
    当通过本地新建多方会议时，“SRTP加密”选择“禁用”或者“最大互通性”，此时召开的为不加密会议，如果远端设置了加密，那么该远端不能成功加入会议。
    • 禁用：对媒体流不加密。若禁用，需设置远端SRTP加密为禁用或最大互通性才能进行视频通话。
    • 启用：对媒体流进行加密。若启用，需设置远端SRTP加密为启用或最大互通性才能进行视频通话。为了确保通信安全，建议选择此项。
    • 最大互通性：以保证能够建立通话为首要条件来决策是否加密。如果远端不加密，则不进行加密限制。
  • 自动接听：启用后可自动接听来电。
  • 演示：启用后可共享桌面给远端。
  • 带宽：设置发起呼叫使用的带宽。当设置为64k bit/s时不能共享演示。
  • 通话心跳：启用后可以在远端会场突然断网的情况下，收到通话挂断通知。

单击“确定”，立即生效。通话过程中不支持呼叫和演示设置。

修改密码

输入原密码，再输入新密码和确认密码，完成密码设置。

修改密码后需要重新登录，使设置的密码生效。
