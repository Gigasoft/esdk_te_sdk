登录/退出

介绍TE Desktop登录前配置、登录和退出的操作方法。

登录前配置

首次使用帐号登录TE Desktop前需要配置注册服务器和升级服务器，注册服务器即SC（SwitchCentre）服务器。配置注册服务器后，TE Desktop可以通过帐号拨打其他注册在同一SC服务器上的终端和加入视讯会议。配置服务器参数方法如下：
  1. 启动TE Desktop。
  2. 在登录界面右下角单击，打开如图1和图2所示界面。

    图1 注册服务器设置
    图2 高级设置

  3. 配置注册服务器和高级设置参数，参数及说明请参见表1。

    表1 注册服务器和高级设置参数

    参数

    说明

    服务器设置参数

    注册服务器地址和端口号
    • 地址：SC服务器的IP地址，代理服务器的IP地址或者域名。
    • 端口号：SC服务器的端口号。

    请从SC服务器管理员处获取。

    SIP URI

    使用“帐号@SIP服务器的IP地址或域名”格式输入。

    帐号为从SC服务器放号的号码。帐号、SIP服务器IP地址或者域名请从SC服务器管理员处获取。

    跳过登录

    		 单击后您将只能使用“IP地址:远端端口号”呼叫远端，并且您将不能加入视讯会议。

    高级设置参数

    本地端口
    • 媒体端口：设置媒体流端口号。
    • SIP端口：设置SIP信令端口号。

    传输模式

    选择SIP信令传输的协议类型。

    • TLS：通过TLS为SIP信令提供加密安全传输。TLS传输模式具有较高安全性，建议选择此项。

      选择此项，需要导入TLS根证书。缺省导入华为TLS根证书，即华为智真系统自带的TLS根证书，您也可以选择其他证书进行导入。

      说明：
      导入证书前，请确认证书的安全性，建议导入安全机构颁发的证书，以免带来风险。
    • UDP：通过UDP为SIP信令提供最佳传输。

    升级服务器地址和端口号
    • 地址：升级服务器的IP地址。
    • 端口号：升级服务器的端口号。

    注册服务器参数配置后立即生效，升级服务器参数需重启生效。

  4. 单击“保存”。

登录

  1. 启动TE Desktop。
  2. 输入帐号和密码。

    未配置注册服务器时登录需要输入服务器地址，如图3所示。帐号为从SC服务器放号的号码，帐号和密码从SC服务器管理员处获取。

    图3 登录界面

  3. 单击，进入TE Desktop主界面。

    说明：
    单击可打开在线帮助页面，以帮助您了解和使用TE Desktop。

退出登录

  • 如果您在登录界面尚未登录TE Desktop，单击右上方的即可退出。
  • 如果您已经登录TE Desktop：
    • 单击主界面上方的，在弹出的下拉框中选择“退出”。
    • 单击主界面右上方的。如果您已经在“ > 基本设置”下选择“退出程序”，且不勾选“关闭主窗口出现提示框”，则单击主界面右上方的可立即退出。
