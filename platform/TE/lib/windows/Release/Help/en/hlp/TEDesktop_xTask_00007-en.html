Technical Specifications

This topic describes the technical specifications of the TE Desktop.

Table 1 TE Desktop technical specifications

Item

Description

Operating system

Windows XP (32-bit), Windows 7 (32- or 64-bit), or Windows 8 (32- or 64-bit) functioning with x86-based hardware

Communication protocols

Session Initiation Protocol (SIP), Real-Time Transport Protocol (RTP), Real-Time Transport Control Protocol (RTCP), and Binary Floor Control Protocol (BFCP)

Video standards

Protocol: H.264 BP, H.264 HP

Resolution: 720P 30 fps for sent images, 1080P 30 fps for received images

Format: VGA, 4CIF, CIF, QVGA, and QCIF.

Presentation standards

Protocol: H.264 BP, H.264 HP

Resolution: 1920 x 1200 5 fps for sent images, 1920 x 1200 10 fps for received images

Format: VGA, 4CIF, CIF, QVGA, and QCIF.

Audio standards

G.711A, G.711U, G.722, and G.729A, and Internet Low Bitrate Codec (iLBC)

Networks

Ethernetand Wi-Fi

Two-stage dialing

Dual tone multiple frequency (DTMF) and RFC 2833

Security protocols

SIP Over Transport Layer Security (TLS), Secure Real-time Transport Protocol (SRTP), FTPS, LDAPS and HTTPS

Bandwidths

64-2048 kbit/s

Audio processing

Acoustic echo cancellation (AEC), automatic gain control (AGC), and automatic noise suppression (ANS)

Languages

English, Chinese Simplified, Chinese Traditional, Russian, French, Portuguese(BR), Portuguese(EU), Spanish, Japanese
