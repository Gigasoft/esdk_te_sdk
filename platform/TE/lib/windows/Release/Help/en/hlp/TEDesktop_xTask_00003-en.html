Logging In and Out

This topic describes how to set required parameters before using the TE Desktop and how to log in to and out of the TE Desktop.

Preparing for Login

If you are using the TE Desktop for the first time, set the registration server (Switch Centre server) and update server parameters. After setting registration server parameters, you can place calls to other videoconferencing endpoints registered with the same registration server using their accounts and join video conferences. To set the parameters, complete the following steps:
  1. Start the TE Desktop.
  2. In the lower right corner of the login window, click to display the tab shown in Figure 1.

    The Advanced Settings tab shown in Figure 2 shows the advanced parameters.

    Figure 1 Registration server parameters

    Figure 2 Advanced parameters

  3. Set the registration and advanced parameters described in Table 1.

    Table 1 Registration and advanced parameters

    Parameter

    Description

    Registration Server

    Registration Server Address and Port
    • Address: SC server IP address or proxy server IP address/domain name.
    • Port: SC server port.

    Obtain the IP address and port from the SC server administrator.

    SIP URI

    Enter a value in format of account@SIP server IP address or domain name.

    The account is a number allocated by the SC server. You can obtain the account and SIP server IP address or domain name from the SC server administrator.

    Skip Login

    Click to log in without entering an account number and password. If you log in this way, you can only place calls by entering IP address:port number and cannot join video conferences.

    Advanced Settings

    Local Port
    • Media Port: Media stream port.
    • SIP Port: SIP signaling port.

    Transmission mode

    Protocol used for Session Initiation Protocol (SIP) calls.

    • TLS: Use the Transport Layer Security (TLS) protocol to fortify transmission security. TLS is recommended due to its high security performance.

      If you select this option, import a TLS root certificate. By default, the Huawei TLS root certificate has been imported. You can also import other root certificates.

      NOTE:
      Before importing a certificate, make sure it is issued by a security authority to prevent security risks.
    • UDP: Use the User Datagram Protocol (UDP) to implement transmission with reduced latency.

    Update Server Address and Port
    • Address: update server IP address.
    • Port: update server port.

    The settings for the registration server take effect immediately, while those for the update server take effect after the TE Desktop restarts.

  4. Click Save.

Login

  1. Start the TE Desktop.
  2. Enter the account number and password, as shown in Figure 3.

    If the registration server is not configured, the server IP address is required during the login to the TE Desktop. Obtain the account number and password from the registration server administrator.

    Figure 3 Login window

  3. Click .

    NOTE:
    Click to access online help regarding the TE Desktop.

Exit

  • If you have not logged in, click in the upper right corner.
  • If you have logged in:
    • In the up left corner of the main window, click and choose Exit from the displayed shortcut menu.
    • Click in the upper right corner of the main window. If Exit Program is selected and Prompt before closing is not selected under > Basic, click to directly exit the TE Desktop.
