Basic Functions

This topic describes the functions of the TE Desktop.

The TE Desktop is a secure, stable videoconferencing software running on PCs. It is used together with the MCU, videoconferencing endpoint, telepresence system, and SC server, and connects to Huawei's telepresence system using Wi-Fi or Ethernet.

The TE Desktop offers the following functions:

  • Joins an audiovisual conference over a Wi-Fi or Ethernet.
  • Supports high quality video conferences using the H.264 baseline profile (BP) and H.264 high profile (HP).
  • Initiates a point-to-point call or multipoint conference using the standard Session Initiation Protocol (SIP).
  • Supports control the conference in a multipoint conference.
  • Supports both local directory and corporate directory, as well as queries and updates the directories using FTPS or LDAPS.
  • Imports and exports the local directory, and adds contacts in the corporate directory to the local directory.
  • Switches a point-to-point call to a multipoint conference passively.
  • Joins a conference simply by entering the conference ID or clicking the conference link in the conference notification email.
  • Switches between audio and video calls.
  • Shares presentation.
  • Supports audiovisual tests.
  • Initiates calls from the contacts list or call records.
  • Turns on or off the local camera in the video call window.
  • Performs online updates using the update server without affecting existing data.
