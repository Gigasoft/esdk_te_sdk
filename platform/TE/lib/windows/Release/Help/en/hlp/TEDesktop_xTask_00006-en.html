Settings

You can configure personal info, system, and advanced settings.

To access the settings window, perform any of the following in the TE Desktop main window:
  • Click in the upper left corner and choose Settings from the displayed shortcut menu.
  • Click in the lower left corner.
  • Click your avatar.

Personal Info Settings

Under personal info, you can view your account, SIP URI, IP address, and port number.

Basic Settings

  • Main Window: Set the display mode of the main window. The settings take effect immediately.
  • Login: Set whether the TE Desktop automatically starts, logs you in when the computer is started, set the default page after login. The settings take effect after you restart the TE Desktop. Users who log in to the TE Desktop without accounts cannot set the auto-login, remember-password, and auto-startup functions.
  • Language: Set the language of the TE Desktop. The settings take effect after you restart the TE Desktop.
  • Ringtone: Set the ringtone for incoming calls. The settings take effect immediately.
    • To select a ringtone provided with the system, select one from the drop-down list box.
    • To play the selected ringtone, click .

Media Settings

  • Audio Input: Select an audio input device from the drop-down list box. Click to test the microphone.
  • Audio Output: Select an audio output device from the drop-down list box. Click to test the speaker.
  • Video Devices: Set video device parameters after installing the video input device.

    Select a video input device from the drop-down list box. Click to test the camera.

    • Better quality: High resolution rate is given priority at the given bandwidth.
    • Smoother playback: High frame rate is given priority at the given bandwidth.

Click OK. The settings take effect immediately. Media settings are not configurable during a call.

Call and Present Settings

  • SRTP Encryption: Set whether to encrypt the media streams to be transmitted.
    NOTE:
    If SRTP Encryption is set to Disable or Max interconnectivity, the multipoint conference initiated from the local site is an unencrypted one. If SRTP Encryption is set to Enable for a remote site, the remote site cannot join the conference.
    • Disable: Media streams are not encrypted. If this option is selected, the other party must set SRTP encryption to disable or maximum interconnectivity to set up a video call.
    • Enable: Media streams are encrypted. If this option is selected, the other party must set Secure Real-time Transport Protocol (SRTP) encryption to enable or maximum interconnectivity to set up a video call.
    • Max interconnectivity: Media streams are encrypted only when a call can be set up. If the other party does not support encryption, no media stream will be encrypted.
  • Auto Answer: The TE Desktop answers incoming calls automatically.
  • Presentation: Set whether you can share the computer desktop as a presentation with remote parties.
  • Bandwidth: Set the bandwidth used for placing calls. When the bandwidth is set to 64 kbit/s, you cannot share the computer desktop.
  • Call Heartbeat: After Call Heartbeat is set to Enable, the TE Desktop will receive a call release notification if the remote site is disconnected from the network.

Click OK. The settings take effect immediately. Call and present settings are not configurable during a call.

Password Change

Enter the old password, new password, and confirm password.

Re-log in to the TE Desktop using the new password for it to take effect.
